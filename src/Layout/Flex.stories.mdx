# Flex

Flex is a simple layout container works as a Flex box.

# Usage

```js
import {Flex} from 'react-native-design-system';

<Flex size="lg">
  <Button>Press Me</Button>
</Flex>;
```

# Props

<Props of={Flex} />

# Space

Customize padding for the child. `space` prop accepts one of the given values: `none`, `xxsmall`, `xsmall`, `small`, `medium`, `large`, `xlarge`, `xxlarge`

<Preview>
  <Story name="space">
    <Flex size="lg">
      <Button>Press Me</Button>
    </Flex>
  </Story>
</Preview>

# Background

You can also pass a `background` color. Color string should be present inside `colors` of theme.

<Preview>
  <Story name="background">
    <Flex background="bg100" space="2xl">
      <Button>Press Me</Button>
    </Flex>
  </Story>
</Preview>
